Commands Usage
  • Display commands. /help
  • Display informations. /info
  • Change the bot's language /setlanguage
  • Change the bot's main color /setcolor
  • Set the Welcome Channel. /setwelcome
  • Set the Logs Channel. /setlogs
  • Set the Support Channel. /setticket
  • Set the Music Commands Channel. /setcommandzik
  • Set the Music Listening Channel. /setlistenzik
  • Set the Birthday Channel. /setbirthday
  • Manage Auto-roles. /autorole
  • Assign a role. /setrole
  • Manage RSS Feeds. /rss
  • (Un)Ban a member. /(un)ban
  • Kick a member. /kick
  • (Un)Mute a member. /(un)mute
  • Create a giveaway. /giveaway
  • (Un)Lock a channel. /(un)lock
  • Purge the channel. /purge
  • Send a message. /msg
  • Send an embedded message. /msgembed
  • Send a webhook message. /webhookmsg
  • Send an embedded message via a webhook. /webhookmsgembed
  • Create a reminder for a new episode. /release
  • Open one's storage. /mydeb
  • Display anime/manga info. $manga/anime Title
  • Display user profile $profil @user/user#0000/userid
  • Assign a network to the profile /resprofil
  • Search for a wallpaper. $wallpaper Title
  • Retrieve an image by type. /pics_v(1/2)
  • Generate a password. /passgen
  • Translate a text. /translate
  • Convert a value from another format. /conv
  • (En/De)crypt a text. /crypt
  • Shorten a URL. /urlshort
  • Generate a QR Code. /genqr
  • Generate an image. /imgen
  • Listen to a song. $zikplay url-youtube
  • Skip the song. $zikskip
  • Stop the song. $zikstop
  • Enter your birthday. /birthday
  • Rolls a virtual die. /die
